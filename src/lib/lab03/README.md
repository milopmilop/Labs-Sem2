<h1 align = 'center'>models.py</h1>

> В models.py реализованны два дочерних класса:
>
> - Penthouse: Дочерний класс, реализующий тип квартиры - пентхаус.
> - Studio: Дочерний класс, реализующий тип квартиры - студию.
>
> Оба класса наследуют от Apartment и имеют свои уникальные атрибуты и методы.


# Дочерний класс Penthouse:

## Атрибуты:

* `address`, `area` и т.д. (унаследовано от Apartment).

* `terrace` - Наличие/Доступ к террасе.

* `ceiling_height` - Высота потолков.

## Методы:

* Дандер-методы (Переопределнные):

  * `__str__()` - Переопределенный метод для строкового представления студии

  * `__repr__()` - Переопределенный метод для представления студии в виде строки

  * `__eq__()` - Переопределенный метод для сравнения студий

* Бизнес-методы:

    * `apply_discount()`, `remove_discount()` - Унаследованные методы для применения и удаления скидки.

    * `terrace_access_switch()` - Метод для переключения доступа к террасе.

    * `calculate_price()` - Переопределенный метод для расчета цены пентхауса.

# Дочерний класс Studio:

## Атрибуты:

* `address`, `area` и т.д. (унаследовано от Apartment).

* `rooms` (Переопределенный атрибут) - Количество комнат (Всегда 1).

* `floor` - Этаж.

## Методы:

* Дандер-методы (Переопределнные):

  * `__str__()` - Переопределенный метод для строкового представления студии

  * `__repr__()` - Переопределенный метод для представления студии в виде строки

  * `__eq__()` - Переопределенный метод для сравнения студий

* Бизнес-методы:

    * `apply_discount()`, `remove_discount()` - Унаследованные методы для применения и удаления скидки.

    * `switch_floor()` - Метод для переключения этажа.

    * `calculate_price()` - Переопределенный метод для расчета цены студии.

<h1 align = 'center'>validate.py</h1>

## Ф-ии валидации атрибутов/методов дочерних классов Penthouse и Studio:

* `validate_floor` - Валидация этажа (type: int, range: [0, 2]).

* `validate_terrace` - Валидация террасы (type: bool).

* `validate_ceiling_height` - Валидация высоты потолков (type: float, range: (0, +∞)).

* `validate_rooms` - Валидация количества комнат (type: int, range: [1, 20]).

* `validate_address` - Валидация адреса (type: str, not empty).

* `validate_area` - Валидация площади (type: float, range: [1, 1000]).

* `validate_price` - Валидация цены (type: float, range: [0, +∞]).

* `validate_discount` - Валидация скидки (type: float, range: [0, 1]).

* `validate_switch_floor` - Валидация переключения этажа (type: int, range: [0, 2]).

* `validate_calculate_price` - Валидация расчета цены (type: float, range: [0, +∞)).
