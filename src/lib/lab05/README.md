<h1 align = 'center'>strategies.py</h1>

> В strategies.py реализованы стратегии сортировки, фильтрации и трансформации в функциональном стиле.

## Сортировка и фильтрация:

* `by_price`, `by_area`, `by_address` - Функции-стратегии для получения ключей сортировки.
 
* `is_available`, `is_expensive` - Функции-предикаты для фильтрации объектов.

## Фабрики функций:

* `make_price_filter` - Фабрика, создающая функцию-фильтр для ограничения по максимальной цене.

## Callable-объекты и Делегаты:

* `SortByPrice` - Объект-стратегия для сортировки.

* `FilterAvailable` - Объект-стратегия для фильтрации.

* `apply_discount` - Фабрика функций-трансформаторов для метода `apply`.

<h1 align = 'center'>collection.py</h1>

> В collection.py представлена обновленная реализация House с методами функциональной обработки и поддержкой цепочек.

## Новые методы:

* `filter_by` - Универсальный фильтр. Принимает предикат и возвращает **новый** объект House.

* `sort_by` - Универсальная сортировка. Принимает функцию-ключ и возвращает **новый** объект House.

* `apply` - Массовое применение функции к элементам. Возвращает `self`.

## Поддержка цепочек (Method Chaining):

Реализована возможность последовательного вызова методов: 
`house.filter_by(...).sort_by(...).apply(...)`.

def for_three():
    print("\n" + "="*50)
    print("GRADE 3: ФУНКЦИИ ВЫСШЕГО ПОРЯДКА (SORTED, FILTER)")
    print("="*50)
    data = get_test_data()
    
    # 1. Сортировка через встроенную функцию sorted() и передачу функций-стратегий
    print("Сортировка по цене (через функцию by_price):")
    for a in sorted(data, key=by_price):
        print(f"   {a.address}: {a.price:,.0f} руб.")
        
    # 2. Фильтрация через встроенную функцию filter()
    print("\nФильтрация: Только доступные (через функцию is_available):")
    for a in filter(is_available, data):
        print(f"   [OK] {a.address}")
```

def for_four():

        print(f"   - {a.address}: {a.price:,.0f} руб.")

def for_five():
    print("\n" + "="*50)
    print("GRADE 5: ЦЕПОЧКИ (CHAINING) И CALLABLE ОБЪЕКТЫ")
    print("="*50)
    
    # Инициализация коллекции
    house = House(get_test_data())
    
    print("Выполнение цепочки операций (Method Chaining):")
    print("filter_by(Доступные) -> sort_by(Цена) -> apply(Скидка 10%)\n")
    
    # Демонстрация Fluent Interface (Grade 5)
    # Используем Callable-объекты из strategies.py
    result = (house
              .filter_by(FilterAvailable()) # Предикат-объект
              .sort_by(SortByPrice())       # Ключ-объект
              .apply(apply_discount(10)))   # Массовое изменение
              
    for a in result.get_all():
        print(f"   Результат: {a.address} | Новая цена: {a.price:,.0f} руб.")
```
